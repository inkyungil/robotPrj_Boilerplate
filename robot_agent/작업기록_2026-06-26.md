### "청소 vs 재매핑" 결론

- 채택: **B안(설정 튜닝)으로 처음부터 깨끗하게** + 필요시 마무리 청소.

### inflation_radius 재조정 (0.08 → 0.05)
- map2에서 우측 두 물체(#3 x71-77,y20-27 / #7 x81-89,y34-41) 최단간격 **약 8칸=16cm**로 찍힘
  (실제는 20cm지만 맵에서 물체가 약간 부풀려져 16cm처럼 표현됨).
- inflation 0.08이면 16−8−8 = **0cm** → 양쪽 inflation이 가운데서 겹쳐 통로 막힘.
- 실제 20cm(로봇 12cm 대비 양쪽 4cm 여유)라 안전 → **inflation_radius 0.08 → 0.05** 로 낮춤
  (nav2_params.yaml local L169 / global L228, src+install 양쪽).

| inflation_radius | 맵상 16cm 기준 | 실제 20cm 기준 |
|---|---|---|
| 0.08 (이전) | 16−8−8 = **0cm** (막힘) | 20−8−8 = 4cm |
| **0.05 (현재)** | 16−5−5 = **6cm 통로** | 20−5−5 = 10cm |

- inflation은 런타임 값 → **navigation 재시작만으로 즉시 반영** (재매핑/재빌드 불필요).

### 속도 조정 (자율주행이 빠름)
